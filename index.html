国税庁CSVのダウンロード手順
  1. 国税庁 法人番号公表サイト を開く
  2. 「全件データ」の最新版をダウンロード（zipファイル・約500MB）
  3. zipを解凍するとCSVが出てくる
  4. 「Step 1」でそのCSVをアップロード
  5. 業種・都道府県を選んでリストを作成
CSVの列構成（国税庁フォーマット）
列1: シーケンス番号 / 列2: 法人番号 / 列3: 処理区分 / 列4: 訂正区分 / 列5: 取消区分 / 列6: 法人種別 / 列7: 法人名 / 列8: 法人名フリガナ / 列9: 法人名英語 / 列10: 郵便番号 / 列11: 都道府県コード / 列12: 市区町村コード / 列13: 都道府県名 / 列14: 市区町村名 / 列15: 丁目番地等 / 列16〜: 業種コード等
補完の仕組み
ブラウザ上でCSVを読み込んで業種・都道府県で絞り込み → 法人番号をサーバーに送信 → gBizINFO APIで電話・FAX・メール・HPを取得 → スプレッドシートに保存。 CSVはサーバーに送信しないためメモリ消費なし。
